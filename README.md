# Weather-App
Weather-App is a website that allows users to search for the weather by inputting the city, state, or country.

Submitted by: Jonathan Chery

Time spent: 1:30 hrs

## Tech(s) Used

The following **technologies** were used to complete this website:

* HTML/CSS
* Bootstrap
* PHP
* JQuery
* Curl
* OpenWeatherMap API
